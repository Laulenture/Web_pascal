7/ Les types d'enregistrements

Un type structuré (ou enregistrement) est une structure de données consistant en un nombre fixé de composants, appelés champs. A la différence du tableau, ces composants ne sont pas obligatoirement du même type, et ne sont pas indexés. La définition du type enregistrement précise pour chaque composant un identificateur de champ, dont la portée est limitée à l'enregistrement, et le type de ce composant.

Syntaxe:

Type nom_du_type = Record
champ1 : type1;
champ2 : type2;
champ3 : type3;
end ;
